# Linux网络

## OSI七层模型
Open System Interconnection,全称呼：```开放系统互连参考模型```

|        模型        |        定义      | 
|    :-:             |      :-:         | 
|       应用层       |    App           |
|       表示层       |                  |
|       会话层       |   SP             |
|      传输层        |   TP             |
|      网络层        | Np               |
|    数据链路层      |         -        |
|    物理层          | -                |
